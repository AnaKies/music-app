## Purpose

This directory contains detailed assignable briefs for every junior task.
The briefs are grouped by feature range so that planning stays navigable while each task remains specific enough for direct assignment.

## Documents

- [Task Briefs F1-F4](./f1-f4.md): Detailed briefs for case entry, interview, case persistence, and upload acceptance.
- [Task Briefs F5-F8](./f5-f8.md): Detailed briefs for parsing, recommendation context, recommendation generation, and recommendation review.
- [Task Briefs F8b](./f8b-score-preview.md): Detailed briefs for safe score preview and result comparison.
- [Task Briefs F9-F12](./f9-f12.md): Detailed briefs for transformation, export, status visibility, and result download.
- [Task Briefs F13-F16](./f13-f16.md): Detailed briefs for case edit/reset, stale handling, retryable failure recovery, and deployment verification.

## Task Brief Template

Each task brief should include:

- task objective
- working context
- expected deliverable
- likely implementation area
- dependencies
- acceptance criteria
- out-of-scope boundary
- suggested verification
