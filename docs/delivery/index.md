# Delivery Index

This directory contains approved delivery-planning documents derived from the architecture.

## Documents

- [Junior Task Breakdown](./junior-task-breakdown.md): Junior-oriented task decomposition of the MVP features into small implementation work packages.
- [Task Briefs Index](./task-briefs/index.md): Detailed assignable briefs for each junior task grouped by feature range.

## Structure Rules

- Delivery documents should derive from approved architecture and implementation plans rather than redefining system boundaries.
- Team allocation and task slicing belong here instead of in architecture overview documents.
- New delivery topics should be added as separate files when they introduce a distinct planning concern.
