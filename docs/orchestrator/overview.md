# Orchestrator Overview

Reference: [Orchestrator Index](./index.md)

## Purpose

This directory stores the approved final-state orchestration view for the project.

## Current Scope

The orchestration documentation maintained here may cover:

- routing rules
- agent authority boundaries
- collaboration sequences
- escalation rules
- conflict resolution patterns

## Documentation Principle

Only approved final orchestration documentation belongs here.
Working notes, rejected variants, and temporary exploration are intentionally excluded.

## Maintenance Rule

The orchestration agent maintains this documentation in parallel with approved coordination and routing changes.
