# Backend Overview

Reference: [Backend Index](./index.md)

## Purpose

This directory stores the approved final-state backend engineering view for the project.

## Current Scope

The backend documentation maintained here may cover:

- backend structure
- implementation stack and runtime tooling
- service responsibilities
- API design and contracts
- persistence behavior
- integration patterns
- background processing
- runtime and observability expectations

## Documentation Principle

Only approved final backend documentation belongs here.
Working notes, rejected variants, and temporary exploration are intentionally excluded.

## Maintenance Rule

The backend agent maintains this documentation in parallel with approved backend changes.
