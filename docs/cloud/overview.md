# Cloud Overview

Reference: [Cloud Index](./index.md)

## Purpose

This directory stores the approved final-state cloud and production-readiness view for the project.

## Current Scope

The cloud documentation maintained here may cover:

- hosting choices
- deployment strategy
- CI/CD patterns
- runtime and platform selection
- infrastructure tooling
- prototype-to-production transitions
- scaling and operations expectations
- MVP deployment topology and managed-service strategy

## Documentation Principle

Only approved final cloud documentation belongs here.
Working notes, rejected variants, and temporary exploration are intentionally excluded.

## Maintenance Rule

The cloud agent maintains this documentation in parallel with approved cloud-related changes.
