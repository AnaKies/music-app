# Safety Index

This directory contains the approved target safety and security state for the project.

## Documents

- [Safety Agent](../../agents/Safety.md): Behavior definition and security rules for the project-specific safety agent.
- [Safety Overview](./overview.md): High-level description of the current safety documentation scope and maintenance rules.
- [MVP Safety Model](./mvp-safety-model.md): Threat boundaries, minimum safeguards, and safe-handling rules for the current MVP.

## Structure Rules

- New safety topics should be documented in dedicated files.
- Existing topic files should be extended when the work goes deeper into the same safety area.
- Large topics should be split into subtopics and linked from this index.
