# Design Overview

Reference: [Design Index](./index.md)

## Purpose

This directory stores the approved final-state design direction for the project.

## Current Scope

The design documentation maintained here may cover:

- visual language
- layout principles
- interaction patterns
- interface copy guidance
- responsive behavior
- accessibility expectations

## Documentation Principle

Only approved final design belongs here.
Working notes, rejected variants, and temporary exploration are intentionally excluded.

## Maintenance Rule

The design agent maintains this documentation in parallel with approved frontend and interface changes.
