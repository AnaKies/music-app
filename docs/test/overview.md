# Test Overview

Reference: [Test Index](./index.md)

## Purpose

This directory stores the approved final-state testing and verification view for the project.

## Current Scope

The test documentation maintained here may cover:

- test strategy
- coverage boundaries
- regression-critical areas
- AI-related evaluation expectations
- safety-sensitive verification flows
- release confidence criteria

## Documentation Principle

Only approved final test documentation belongs here.
Working notes, rejected variants, and temporary exploration are intentionally excluded.

## Maintenance Rule

The test agent maintains this documentation in parallel with approved testing-related changes.
