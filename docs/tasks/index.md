# Tasks Index

This directory contains approved task-planning documents derived from the architecture.

## Documents

- [Overview](./overview.md): Task-oriented decomposition of the MVP features into small implementation work packages.
- [Task Briefs Index](./task-briefs/index.md): Detailed assignable briefs for each junior task grouped by feature range.
- [Repository Structure](../architecture/repository-structure.md): Recommended repository layout that keeps junior implementation work separated by runtime area and feature concern.

## Structure Rules

- Task documents should derive from approved architecture and implementation plans rather than redefining system boundaries.
- Team allocation and task slicing belong here instead of in architecture overview documents.
- New task-planning topics should be added as separate files when they introduce a distinct planning concern.
