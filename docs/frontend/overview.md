# Frontend Overview

Reference: [Frontend Index](./index.md)

## Purpose

This directory stores the approved final-state frontend implementation view for the project.

## Current Scope

The frontend documentation maintained here may cover:

- frontend structure
- component responsibilities
- page-level composition
- UI state handling
- interaction implementation rules
- UI-layer integration patterns

## Documentation Principle

Only approved final frontend documentation belongs here.
Working notes, rejected variants, and temporary implementation exploration are intentionally excluded.

## Maintenance Rule

The frontend agent maintains this documentation in parallel with approved frontend changes.
